# DnOetel
WEBJS assessment blok 7

## Prerequisites
- NodeJS / NPM

## Getting Started
1. Clone the project
2. Run `npm install`
3. Copy `.env.example` and rename to `.env`
4. Fill in your credentials for api usage
5. Run `npm run build`
6. Enjoy! :tada:
